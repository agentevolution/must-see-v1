<strong>INSTALL</strong>
1. Upload the Must See child theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Must See theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

Please visit <a href="http://www.agentevolution.com/guide/must-see-theme-setup-guide/">http://www.agentevolution.com/guide/must-see-theme-setup-guide/</a> for a setup guide.
